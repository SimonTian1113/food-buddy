# 城市配置

每个城市根据区域自动切换平台权重体系和搜索策略。6 位探员直接处理所有城市，不需要城市代理人。

---

## 区域分类

| 区域 | 搜索引擎地区 | 平台权重体系 |
|------|-------------|-------------|
| 🇭🇰 港澳 | Bing HK (`zh-HK`) | 港澳模式 |
| 🇨🇳 内地 | Bing CN (`zh-CN`) | 内地模式 |
| 🇹🇼 台湾 | Bing TW (`zh-TW`) | 港澳模式（OpenRice 为主） |
| 🌏 海外 | Bing 国际 | 海外模式 |

### 港澳模式平台权重
| 平台 | 权重 | 理由 |
|------|------|------|
| OpenRice | 30% | 本地食客真实评价，本地口碑最可信 |
| Google Maps | 25% | 全球通用，评分相对客观 |
| 大众点评 | 20% | 内地游客视角 |
| 小红书 | 15% | 营销属性强，谨慎采信 |
| TripAdvisor | 10% | 游客偏差大 |

### 内地模式平台权重
| 平台 | 权重 | 厂商 | 理由 |
|------|------|------|------|
| 大众点评 | 20% | 美团 | 消费评价最全，但刷评风险 |
| 抖音 | 20% | 字节 | 视频种草+本地生活，营销密度高 |
| 小红书 | 20% | 小红书 | 种草内容多，"火"≠"好" |
| 高德 | 20% | 阿里 | 本地生活到店入口，地图评价+商户数据 |
| 综合 | 20% | 新闻/旅游/微博 | 第三方视角补充 |

### 海外模式平台权重
| 平台 | 权重 | 理由 |
|------|------|------|
| Google Maps | 40% | 最通用的评分体系 |
| TripAdvisor | 30% | 游客评价最集中 |
| 本地平台 | 20% | 因地制宜（Tabelog/OpenRice 等） |
| 小红书 | 10% | 华人游客视角，营销过滤 |

## 🇭🇰 港澳

### 香港 (hongkong)

- **货币**：HKD
- **人均参考**：小食 30-60 / 普通 80-200 / 高端 300-800
- **区域**：港澳模式（OpenRice 权重最高）
- **本地知识要点**：
  - 中环/上环 = 金融区贵价 + 历史老店
  - 旺角/深水埗 = 本地人地盘，性价比高
  - 尖沙咀/铜锣湾 = 游客+购物区 mixed
  - 湾仔 = 过渡地带，新旧并存

### 澳门 (macau)

- **货币**：MOP（澳门元），港币通用
- **人均参考**：小食 30-60 / 普通 80-200 / 葡国菜 200-500 / 高端 500-1000
- **区域**：港澳模式（OpenRice 权重最高）
- **本地知识要点**：
  - 澳门半岛 = 老城区，地道老店集中
  - 氹仔 = 赌场+游客区，价格虚高
  - 路环 = 安静海边，正宗葡国菜在这
  - 官也街 = 游客打卡街，东西贵但质量一般

## 🇨🇳 内地

### 北京 (beijing)

- **货币**：CNY (¥)
- **人均参考**：小吃 20-50 / 普通 80-200 / 高端 300-800
- **区域**：内地模式（大众点评/抖音/小红书/高德各20%）
- **本地知识要点**：
  - 簋街 = 宵夜一条街，但游客多，本地人去的不多
  - 南锣鼓巷 = 纯游客打卡，吃的贵且一般
  - 牛街 = 清真美食聚集地，本地人认可
  - 胡同小馆 = 别被"胡同"二字忽悠，很多是网红包装

### 上海 (shanghai)

- **货币**：CNY (¥)
- **人均参考**：小吃 20-60 / 普通 100-250 / 高端 400-1000
- **区域**：内地模式
- **本地知识要点**：
  - 南京路/城隍庙 = 游客区，价格虚高
  - 静安/法租界 = 精致餐饮集中，但性价比参差
  - 本地人吃本帮菜去老街区，不去商场
  - 小红书上"魔都必吃"水分大，要交叉验证

### 广州 (guangzhou)

- **货币**：CNY (¥)
- **人均参考**：早茶 50-120 / 普通 80-200 / 高端 300-600
- **区域**：内地模式
- **本地知识要点**：
  - 北京路/上下九 = 游客区，老店有但比例低
  - 番禺/白云 = 本地人觅食区，性价比高
  - 早茶别去网红店，去老牌酒楼
  - "广州人唔骗广州人"——但网红店会

### 深圳 (shenzhen)

- **货币**：CNY (¥)
- **人均参考**：小吃 20-50 / 普通 80-180 / 高端 300-800
- **区域**：内地模式
- **本地知识要点**：
  - 华强北/福田 = 快餐为主，性价比看运气
  - 南山 = 新派餐饮多，但网红密度高
  - 罗湖老区 = 老店集中，但更新慢
  - 深圳是移民城市，"本地人认可"参考价值不如广佛

### 成都 (chengdu)

- **货币**：CNY (¥)
- **人均参考**：小吃 15-40 / 普通 60-150 / 高端 200-500
- **区域**：内地模式
- **本地知识要点**：
  - 宽窄巷子 = 纯游客，价格翻倍味道减半
  - 锦里 = 同上，打卡可以，吃别去
  - 建设路 = 本地人宵夜，性价比高
  - 火锅别去"网红连锁"，本地人有自己的清单

### 杭州 (hangzhou)

- **货币**：CNY (¥)
- **人均参考**：小吃 20-50 / 普通 80-200 / 高端 300-600
- **区域**：内地模式
- **本地知识要点**：
  - 西湖景区内 = 游客价，性价比低
  - 河坊街 = 打卡街，吃的一般
  - 本地人吃面去老小区附近
  - 龙井村茶馆 = 体验可以，但别指望吃好

### 长沙 (changsha)

- **货币**：CNY (¥)
- **人均参考**：小吃 15-40 / 普通 50-120 / 高端 200-400
- **区域**：内地模式
- **本地知识要点**：
  - 太平街/坡子街 = 游客打卡街，价格高味道一般
  - 文和友 = 打工体验可以，吃就算了
  - 本地人吃宵夜去四方坪/马王堆
  - 长沙是"网红城市"重灾区，交叉验证特别重要

### 重庆 (chongqing)

- **货币**：CNY (¥)
- **人均参考**：小吃 15-40 / 普通 60-150 / 高端 200-400
- **区域**：内地模式
- **本地知识要点**：
  - 洪崖洞 = 纯打卡，吃的又贵又一般
  - 解放碑 = 游客区，附近有老店但要看准
  - 南岸/沙坪坝 = 本地人火锅聚集地
  - 重庆火锅"排队王"未必是最好吃的

### 武汉 (wuhan)

- **货币**：CNY (¥)
- **人均参考**：过早 10-30 / 普通 50-120 / 高端 200-400
- **区域**：内地模式
- **本地知识要点**：
  - 户部巷 = 游客打卡，本地人不去
  - 过早（早餐）文化极强，老社区才是宝藏
  - 万松园 = 本地人认可的美食街
  - 热干面别去网红店，去小区门口的

### 南京 (nanjing)

- **货币**：CNY (¥)
- **人均参考**：小吃 15-40 / 普通 60-150 / 高端 200-500
- **区域**：内地模式
- **本地知识要点**：
  - 夫子庙/老门东 = 游客区，价格高
  - 鸭血粉丝汤 = 街边老店 > 商场网红店
  - 盐水鸭 = 别在景区买真空包装的
  - 本地人吃小龙虾去江宁/浦口

### 西安 (xian)

- **货币**：CNY (¥)
- **人均参考**：小吃 10-30 / 普通 50-120 / 高端 200-400
- **区域**：内地模式
- **本地知识要点**：
  - 回民街 = 纯游客，价格翻倍
  - 本地人去洒金桥/大皮院
  - 肉夹馍 = 汉民腊汁肉夹馍 vs 回民腊牛肉夹馍，不一样
  - 泡馍 = 别去网红店，去老字号

## 🇹🇼 台湾

### 台北 (taipei)

- **货币**：TWD (NT$)
- **人均参考**：小吃 50-150 / 普通 200-500 / 高端 800-2000
- **区域**：港澳模式（OpenRice 为主）

## 🌏 海外

### 曼谷 (bangkok)

- **货币**：THB (฿)
- **人均参考**：路边摊 50-150 / 普通 200-500 / 高端 800-2000
- **区域**：海外模式（Google Maps 权重最高）
- **本地知识要点**：
  - 考山路 = 背包客区，性价比低
  - Thong Lor = 精致餐饮区，但价格偏高
  - 街边摊 = 曼谷的灵魂，但卫生要留意
  - 米其林街头店值得专门去

### 东京 (tokyo)

- **货币**：JPY (¥)
- **人均参考**：立食 500-1500 / 居酒屋 3000-6000 / 高端 10000-25000
- **区域**：海外模式（Google Maps 权重最高）
- **本地知识要点**：
  - Tabelog 是日本本地最权威的评分平台
  - 涩谷/原宿 = 年轻人区，网红店多
  - 筑地/丰洲 = 海鲜，但游客密度高
  - 居酒屋 = 车站旁小巷里的比商业街的好

### 首尔 (seoul)

- **货币**：KRW (₩)
- **人均参考**：小食 8000-15000 / 普通 15000-35000 / 高端 50000-120000
- **区域**：海外模式（Google Maps 权重最高）
- **本地知识要点**：
  - 明洞 = 纯游客区，价格虚高
  - 弘大 = 年轻人区，网红密度高
  - 广藏市场 = 传统市场，但游客越来越多
  - 本地人吃烤肉去江南/驿三
